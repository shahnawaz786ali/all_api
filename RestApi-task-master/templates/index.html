List of API/Endpoints:

  • To create Category and sub-category : http://localhost:8000/add-category

  • To Get all Product : http://localhost:8000/get-product-data

  • To Create Product : http://localhost:8000/create-product-data

  • To Edit Product : http://localhost:8000/edit-product-data/ Note: Specify Id to be edited in url

  • To Delete product : http://localhost:8000/del-product-data/ Note: Specify Id to be deleted in url

  • To send excel file using celery : http://localhost:8000/send-excel-file

  • To Get List of all product using Redis caching : http://localhost:8000/list-of-product

  • To Schedule Email : http://localhost:8000/schedule-email
